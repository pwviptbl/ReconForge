# Visualizador de Histórico da IA

## Comandos Disponíveis

```bash
# 1. Listar sessões disponíveis
python3 historico_ia/visualizar_historico.py listar 5

# 2. Analisar sessão específica
python3 historico_ia/visualizar_historico.py sessao pentest_inteligente_20250828_165929

# 3. Ver primeira decisão da IA
python3 historico_ia/visualizar_historico.py interacao pentest_inteligente_20250828_165929 1

# 4. Ver última decisão da IA
python3 historico_ia/visualizar_historico.py interacao pentest_inteligente_20250828_165929 20

# 5. Gerar relatório analítico
python3 historico_ia/visualizar_historico.py relatorio pentest_inteligente_20250828_165929
```
